# Content & Growth Strategy

## Flywheel Growth → Sales

```
Contenido técnico (demonstra expertise)
  → Lead inbound chega (form, social, referral)
  → Mari qualifica fit (empresa tem problema que resolvemos?)
  → Issue no Multica: 'Lead: [Empresa]'
  → Bi (SDR) recebe e inicia outreach warm
```

## Tipos de Conteúdo

1. **Técnico profundo** — arquitetura, postmortems, ADRs (atrai engenheiros)
2. **Thought leadership** — visões sobre agent orchestration, memory OS (atrai CTOs)
3. **Case studies** — como cliente X resolveu problema Y (atrai decisores)
4. **Social** — Twitter/X curation via Growth Agent (atrai comunidade)

## Métricas

- Tráfego orgânico mensal
- Conversão visitante → lead
- Lead → demo (taxa de qualificação)
- CAC orgânico (quase zero se bem feito)
