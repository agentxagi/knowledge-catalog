# Sales Outbound Process

## Prospecção (Bi)

Cada mensagem é única. Cada empresa é única. Spam é pra quem não pesquisa.

**Antes de contactar:**
1. Site, LinkedIn, blog, notícias recentes
2. Stack técnica (quando aplicável)
3. Identificar a dor: onde a Valor Digital ajuda?

**Mensagem:**
- Referência específica à empresa
- Pergunta que revela dor
- CTA claro (15 min, não "vamos conversar")

## Qualificação (Rafa)

**BANT simplificado:**
- **B**udget: tem orçamento para ferramenta?
- **A**uthority: fala com decisora?
- **N**eed: tem dor real?
- **T**iming: quando precisa resolver?

Se 3/4 → demo. Se <3 → nurture.

## Demo

Não é feature tour. É "como isso resolve a sua dor específica".
